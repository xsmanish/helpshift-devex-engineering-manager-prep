# 02 — Technical Preparation

## What Is Covered

This section covers the full breadth of technical topics relevant to a Developer Productivity & Platform Engineering Manager role at Helpshift. Topics include Platform Engineering fundamentals, DORA metrics, CI/CD, Kubernetes/EKS, GitOps, AWS cloud engineering, SRE, Observability, DevSecOps, system design scenarios, and a curated technical question bank.

## Learning Objectives

- Master the core concepts of Developer Productivity and Platform Engineering.
- Be able to discuss DORA and Space metrics with depth.
- Understand CI/CD pipelines, build systems, and release engineering best practices.
- Articulate Kubernetes, EKS, and GitOps operational patterns.
- Demonstrate AWS cloud platform engineering knowledge.
- Understand SRE principles, observability stacks, and DevSecOps practices.
- Solve system design scenarios relevant to platform teams.
- Prepare and present your own project stories with impact.
- Practice with a comprehensive technical question bank.
